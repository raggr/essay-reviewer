Privacy & Data Policy

What we collect (if you opt in)

If you tick the consent box before submitting, we store your essay text and the full review we generate, along with metadata: word count, detected genre, writer level, model used, and token count. We never store your API key.

Why we collect it

Stored essays and reviews are used solely to improve this tool — to understand what kinds of writing it handles well or poorly, and to refine the feedback it produces. We do not use them for any other purpose.

Who can see it

Data is stored in a private Supabase database accessible only to the tool's developer. It is never sold, shared with third parties, or used for advertising.

Anthropic's role

Your essay is sent to Anthropic's API to generate the review, regardless of whether you opt in to storage. Anthropic does not train on API inputs by default — see anthropic.com/privacy for their policy.

Opt-in is genuinely optional

If you leave the consent box unticked, your essay is processed and then discarded. Nothing is stored beyond what appears on your screen.

Deletion requests

If you submitted an essay and would like it removed, contact us and we will delete it promptly. Because essays are stored without any account or identifier, you will need to provide a short excerpt so we can locate the record.

Sensitive material

If your essay contains personal, confidential, or unpublished work you would not want stored, please leave the consent box unticked.

How to get an API key

1
Go to console.anthropic.com and create a free account.
2
Once logged in, click API Keys in the left sidebar, then Create Key. Give it any name you like.
3
Copy the key (it starts with sk-ant-) and paste it into the field here. You only need to do this once per browser session.
4
You'll need to add credit on the Billing page — $5 goes a long way. A single review typically costs $1.00–1.30.
Your key is sent directly to Anthropic and is never stored by this tool. Each review is charged to your own account at cost, with no markup.
